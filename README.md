# Exercício 1 — Tempo de Percurso em Vetor

Fazer uma aplicação Java em Eclipse que tenha uma operação que receba um vetor de 1000 posições inteiras, preenchendo todas as posições com valor 0, e gere o tempo gasto, em segundos, para percorrer o vetor.

Repetir o procedimento para vetores de 10000 e 100000 posições.

A aplicação deve ter:
- Uma classe de controle com métodos para operações;
- Uma classe de visão que instancie a classe de controle para a comunicação.
